# BPI JAPAN website

## 中身
- `index.html`：ホームページ本体
- `style.css`：デザイン調整
- `script.js`：動画の音声ボタン・スクロール演出
- `assets/bpi_cm_hero.mp4`：トップ用動画
- `assets/bpi_logo_horizontal.png`：ロゴ
- `assets/bpi_logo_endcard_16x9.png`：SNSやOGP用のロゴ画像
- `assets/bpi_poster.jpg`：動画読み込み前の表示画像

## GitHub Pagesに反映する方法
1. ZIPを解凍
2. 中のファイル一式を、いま公開しているリポジトリの同じ階層に上書きアップロード
3. `index.html` がリポジトリ直下にある状態で保存
4. 数分待ってページを再読み込み

## メモ
スマホでは自動再生の仕様上、音あり自動再生は基本できません。最初はミュートで再生され、ボタンを押すと音ありで再生できます。
